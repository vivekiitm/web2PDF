Save as PDF Updated

Thank you for using the Save as PDF extension!

Updates in this version:
  • The extension now uses a new improved PDF converter.
    • If you have the Pdfcrowd Browser License
      1. Go to the settings dialog.
      2. Select the "New Converter" option.
      3. Click the "Save Settings" button.
    • If you don't have the license, the new PDF converter will be used automatically.

For more info about the extension please visit the homepage.
