Options

The Save as PDF add-on uses the online Pdfcrowd service to create PDFs from web pages. The add-on is free to use. For more info about the add-on please visit the add-on homepage.

If you want to customize generated PDFs, you can buy the Pdfcrowd Browser License:
  1. Sign up for the Browser License
  2. Customize the settings on the Pdfcrowd web site
  3. Click the Save Settings button
